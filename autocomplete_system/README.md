# Autocomplete System

A simple Python project demonstrating a basic autocomplete system implemented using a trie data structure.

## Files

- `autocomplete.py` - core logic for inserting words and suggesting completions.
- `main.py` - example interactive script showing how to use the autocomplete system.

## Running

1. Navigate to the project directory:
   ```bash
   cd c:\Users\nrrak\OneDrive\Desktop\python_projects\autocomplete_system
   ```
2. Run the main script:
   ```bash
   python main.py
   ```
3. Enter prefixes to see suggestions. Type `exit` to quit.

## Extending

You can load words from a file by using the `load_words` helper in `autocomplete.py` or integrate the system into other applications.
